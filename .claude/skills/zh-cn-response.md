# 中文回复技能

## 核心规则

在本项目（智能音箱_无实物 / 网页端AI语音交互中枢）中，**所有 AI 回复必须使用简体中文**。

## 适用范围

| 范围 | 语言要求 |
|------|----------|
| 技术讨论、方案分析、问题解答 | 简体中文 |
| 代码注释 | 简体中文 |
| 文档编写（包括 .md、.txt） | 简体中文 |
| 变量名、函数名、类名 | 英文（保持代码规范） |
| 提交信息（git commit message） | 简体中文 |
| API 字段名、数据库字段 | 英文（保持互操作性） |
| 用户交互界面文本 | 简体中文 |

## 回答风格

- 技术解释用简洁直白的中文，避免翻译腔
- 代码块中的注释用中文
- 技术术语首次出现时可附带英文原名，如"语音识别（ASR）"
- 保持专业、准确，但不失亲切

## 项目上下文

本项目是纯软件实现的智能音箱系统，核心特征：
- 无物理硬件，全部在个人电脑上运行
- 网页端作为交互入口
- 三条处理路径：小爱设备控制 / 大模型对话 / Claude Code 编程执行
- 统一 TTS 语音播报出口

## 触发条件

此 skill 对本项目内所有对话自动生效。
