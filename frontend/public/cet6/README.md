## 放入文件

将每套卷子的 PDF、MP3 文件放入对应目录即可。目录名格式：`年份-月份-套号`

每种文件：
- `paper.pdf` —— 真题试卷（**必需**，否则不会被识别）
- `listening.mp3` —— 听力音频（可选）
- `answers.pdf` —— 答案解析（可选，PDF 格式）

## 自动识别

不需要手动编辑索引文件。
